# gbif_maps
R script for generating gbif style maps

## Overview
> – Currently set up for Nort America and yellow monkeyflower species (<i>Mimulus</i> syn. <i>Erythranthe</i>).
> – Can be customized to desired geographic extent and taxa.
> – Taxon keys can be pulled directly from [gbif](https://www.gbif.org/) URLs or with the `name_backbone` function in the `rgbif` package.
